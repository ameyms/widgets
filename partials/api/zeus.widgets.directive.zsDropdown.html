zsDropdown
directive in module zeus.widgets

Description

A directive for rendering an animated dropdown menu with option for single or multiple selection

Usage

as element:
<zs-dropdown
       list-items="{Array.<object>}"
       [on-item-click="{function($selectedItem)}"]
       [update-menu-label="{boolean}"]
       [label-generator="{function($item)}"]
       [id-generator="{function($item)}"]
       [multi-select="{boolean}"]
       [default-label="{string}"]
       [field-label="{string}"]>
</zs-dropdown>
as attribute
<ANY zs-dropdown
     list-items="{Array.<object>}"
     [on-item-click="{function($selectedItem)}"]
     [update-menu-label="{boolean}"]
     [label-generator="{function($item)}"]
     [id-generator="{function($item)}"]
     [multi-select="{boolean}"]
     [default-label="{string}"]
     [field-label="{string}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.
