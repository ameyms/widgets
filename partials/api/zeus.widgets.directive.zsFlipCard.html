zsFlipCard
directive in module zeus.widgets

Description

Usage

as element:
<zs-flip-card
       [hide-flip-button="{boolean}"]
       [active-face="{string}"]>
</zs-flip-card>

Parameters

ParamTypeDetails
hideFlipButton
(optional)
boolean

Explicitly hide flipping button. Flips can only be done programmatically.

activeFace
(optional)
string

Face that is currently shown or to be shown. Two way bound.
