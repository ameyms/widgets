zsSimpleToggle
directive in module zeus.widgets

Description

This provides the toggle componant which can be used as checkbox and customize with color, click callback etc.

Usage

as element:
<zs-simple-toggle
       [text="{string}"]
       [state="{boolean}"]
       [lighten="{Number}"]
       [switch-color="{string}"]
       [inactive="{boolean}"]
       [zs-change="{function()}"]>
</zs-simple-toggle>

Directive info

  • This directive creates new scope.

Parameters
