zsCheckbox
directive in module zeus.widgets

Description

A directive that provides custom checkbox of varying size.

Usage

as element:
<zs-checkbox
       [item-value="{boolean}"]
       [size="{string}"]
       [on-value-change="{function}"]>
</zs-checkbox>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
itemValue
(optional)
boolean

Current state of checkbox.

size
(optional)
string

Size of the checkbox(small, medium or large).

onValueChange
(optional)
function

Callback method to be executed every time the state of checkbox changes.
